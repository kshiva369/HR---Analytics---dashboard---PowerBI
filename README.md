# HR Analytics Dashboard (Power BI)

This project presents an interactive **HR Analytics Dashboard** built using **Power BI** to analyze workforce data and generate meaningful HR insights.

## 📊 Project Overview
The dashboard helps organizations understand employee trends, workforce distribution, job satisfaction, and hiring patterns using clear visualizations and KPIs.

## 🔑 Key Insights
- Total Employees
- Active Employees
- Average Employee Experience
- Hiring Trends by Year
- Employee Status Distribution
- Department-wise Employee Analysis
- Job Satisfaction by Role
- Top 10 Employees by Salary
- Work Mode Analysis (Remote vs Office)

## 🛠 Tools Used
- Power BI
- Excel
- Data Visualization
- DAX

## 🎯 Purpose
The goal of this project is to transform raw HR data into meaningful insights that support better workforce decision-making.
